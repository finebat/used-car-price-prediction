# used-car-price-prediction
This project focuses on estimating the resale value of used cars using predictive modeling. The analysis began with exploratory data analysis in Python to understand price distributions, identify skewed variables, and detect extreme values that could affect model performance. Since selling price, present price, and kilometers driven showed strong right skew, log transformations were applied to improve linearity and stabilize relationships between variables. Unrealistic outliers were also reviewed and handled to ensure the dataset reflected realistic resale conditions.

Additional feature engineering was performed by deriving car age from the manufacturing year and encoding categorical variables such as fuel type, transmission type, and seller category. A linear regression model was then implemented in Python using scikit learn to estimate used car resale prices based on factors such as market value, vehicle age, and mileage. The model achieved strong predictive performance with an R square of 0.94 and an RMSE of approximately INR 1.2 lakh on the test set, demonstrating how structured data preparation and predictive modeling can support realistic price estimation in the used car market.
